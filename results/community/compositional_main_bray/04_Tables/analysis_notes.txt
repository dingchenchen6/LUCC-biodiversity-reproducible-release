Canonical source script:
scripts/04_community/02_compositional_main_rewrite.R and scripts/03_main_models/01_run_lu_climate_models_main.R

This rewrite now shows Bray-Curtis, Jaccard, and Sorensen in parallel.
Each metric is displayed for total, turnover, and gradient or nestedness components in combined multi-panel figures.

Public-release note:
- In the lightweight GitHub release, source_models/*.rds may be omitted to reduce repository size.
- When those model objects are absent, the coefficient forest is rebuilt from the precomputed fixed-effect table instead.

Support warning:
- Urban climate support is narrow.
- Predictions should be interpreted only within the empirical pair_clim_mean range of each land-use type.
